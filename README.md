# Short Description
This is a warehouse management website for electronics and It has both client side and server side. REST routes are used to make communications in them. Here, Users can manage Items or Inventories and manage the stocks of those items.

# About Website Features
- User can Login, Logout, signup, select an Inventory and Edit it
- Add Inventories and See only your own items
- Manage Inventories
- It is fully Responsive
- Has Forgot Pasword
- After Signup can do Verification
- Has Protected Route for Items management
- User Friendly 

# Technologies Used
- Css -Tailwind Css
- Ant Design for components customization
- Firebase For Authentication
- JWT is used
- React js as Library
- React npm Packages such as router-Dom, react-toastify and So on
